# IronClad IP

> Defend Your Ideas

AI patent intelligence — prior art search, application drafting, portfolio management, office action responses.

![Screenshot placeholder](https://via.placeholder.com/1200x630/0a0a0f/eab308?text=IronClad+IP)

## Built for Claude Mythos

This landing page was built as part of The Fire Dev's moonshot product suite.

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Outputs to `out/` directory (static export, ready for Netlify/Vercel).

## Features

- **Prior Art Search** — Search 180M+ patents across 50+ patent offices
- **Draft Generation** — Complete patent applications from invention disclosures
- **Portfolio Tracking** — Manage deadlines, fees, and licensing opportunities
- **Office Action Response** — AI-drafted responses to USPTO office actions
